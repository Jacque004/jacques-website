Veille Technologique

Un sujet que je suis pour rester à jour sur les évolutions de mon métier.

L’intelligence artificielle dans le développement logiciel

Je m’intéresse à l’impact de l’IA sur le développement web et logiciel : assistants de code (GitHub Copilot, Cursor, ChatGPT), génération et complétion de code, revue de code automatisée, et évolution des pratiques. L’objectif est de suivre les outils et les usages pour les intégrer de façon pertinente tout en gardant la maîtrise du code et de l’architecture.

Points suivis : gains de productivité et limites des LLM, sécurité et qualité du code généré, positionnement du développeur (création, conception, maintenance) et bonnes pratiques (prompting, revue humaine, tests).
